# movieLand app

MovieLand app was developed to complete a code assignment for the position of React Native developer at 24i. This project was developed using React Native and Typescript. It is compatible with both iOS and Android platforms.

## Setup

Use the command yarn setup to install dependencies.

```bash
yarn setup
```

## Usage

```python

# To run on iOS
yarn ios

# To run on android
yarn android

# To run unit tests - currently only one unit test was developed
yarn test
```
